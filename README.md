Ceci est un projet Next.js initialisé avec create-next-app.
Démarrer

Tout d'abord, lancez le serveur de développement :

npm run dev
# ou
yarn dev
# ou
pnpm dev

Ouvrez http://localhost:3000 dans votre navigateur pour voir le résultat.

Vous pouvez commencer à modifier la page en éditant le fichier app/page.js. La page se mettra à jour automatiquement à mesure que vous modifiez le fichier.

Ce projet utilise next/font pour optimiser et charger automatiquement Inter, une police Google personnalisée.
