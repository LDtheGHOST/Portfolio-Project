# Roadmap LD Comedy - 31 mai au 5 juillet 2025

## SPRINT 1 : 31 mai - 7 juin 2025
Objectif : Finalisation du système de rôles et profils

### Semaine 1 (31 mai - 7 juin)
- Mise en place du système de rôles (ARTIST, THEATER, AGENCY)
- Création des modèles Prisma finaux
- Configuration des routes d'API pour la gestion des rôles
- Tests des formulaires d'inscription avec sélection de rôle

Livrables :
- Schéma Prisma complet
- API routes fonctionnelles
- Formulaire d'inscription multi-étapes

## SPRINT 2 : 8 juin - 14 juin 2025
Objectif : Développement des profils différenciés

### Semaine 2 (8-14 juin)
- Création des interfaces de profil selon le rôle
- Implémentation des formulaires de mise à jour de profil
- Système de validation des données
- Mise en place des dashboards spécifiques

Livrables :
- Pages de profil responsive pour chaque rôle
- Système de mise à jour de profil
- Documentation des composants

## SPRINT 3 : 15 juin - 21 juin 2025
Objectif : Système d'upload média et galerie

### Semaine 3 (15-21 juin)
- Intégration de l'upload de vidéos/photos
- Création de la galerie média responsive
- Gestion des miniatures
- Optimisation des médias

Livrables :
- Système d'upload fonctionnel
- Galerie média interactive
- Documentation technique

## SPRINT 4 : 22 juin - 28 juin 2025
Objectif : Page de découverte et recherche

### Semaine 4 (22-28 juin)
- Développement de la page de listing des artistes
- Implémentation des filtres de recherche
- Système de pagination
- Interface de recherche avancée

Livrables :
- Page de recherche complète
- Filtres fonctionnels
- Tests d'intégration

## SPRINT 5 : 29 juin - 5 juillet 2025
Objectif : Finalisation et préparation RNCP

### Semaine 5 (29 juin - 5 juillet)
- Tests complets de l'application
- Correction des bugs
- Documentation finale
- Préparation de la présentation RNCP

Livrables finaux :
1. Application complète avec :
   - Système de rôles fonctionnel
   - Profils différenciés
   - Upload et gestion des médias
   - Système de recherche et filtres
   - Interface responsive

2. Documentation technique :
   - Architecture du projet
   - Schéma de base de données
   - Guide d'utilisation
   - Documentation API

3. Dossier RNCP :
   - Présentation du projet
   - Documentation technique
   - Captures d'écran
   - Guide de déploiement

Technologies clés utilisées :
- Next.js 15 avec TypeScript
- MongoDB + Prisma
- NextAuth.js
- Tailwind CSS
- Framer Motion
- Cloudinary/Vercel Blob pour les médias

Cette roadmap est optimisée pour une présentation RNCP début juillet 2025, avec des livrables concrets à chaque étape et une documentation complète pour l'évaluation.
